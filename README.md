### 思路
首先要明确一点，图像之所以能够不断移动，是因为屏幕在不断刷新，当频率达到一定程度时，我们由于视觉暂留，会认为图像是一个流畅的播放过程。
这里根据原作者思路，将整个页面划分为大小为`25 x 25`的网格，小蛇移动的范围在`(25,75)->(850,600)`
